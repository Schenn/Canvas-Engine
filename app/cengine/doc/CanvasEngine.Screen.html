capture()

Capture a screenshot This method draws the stack of canvases to a single canvas. Then it converts that images into a base64 and sets it as an anchor tag ref with a download attribute Then it triggers a click onto that anchor tag, before removing it. This triggers the browser to (usually) perform a save-as for the image with the filename of the download attribute.
